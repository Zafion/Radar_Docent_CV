Política de Privacidad

Resumen de tratamiento de datos técnicos del sitio.

Esta web no requiere registro de usuarios para consultar la información publicada.

Si el usuario activa funciones opcionales del navegador, como la geolocalización o las alertas push, el sitio puede tratar datos técnicos necesarios para prestar dichas funciones.

En el caso de las alertas push, se almacena la suscripción técnica del navegador para poder enviar avisos generales de nuevas publicaciones.

La geolocalización solo se utiliza si el usuario concede permiso expresamente en su navegador. La web no necesita una cuenta de usuario para esta función.

Para cualquier consulta relacionada con privacidad, puedes contactar en {{ project_email }}.
